# What Time Today

A simple tool to help you send someone your calendar availability.

Just mark your time availability on the calendar, and click copy.

Open an [issue](https://github.com/jasonlearst/what-time-today/issues) for questions/concerns/feature requests.

### Run locally:
```
$ yarn
$ yarn dev
```

### Deploy:
Deployed to GitHub Pages.

```
$ yarn deploy
```
